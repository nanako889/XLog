# XLog


轻量级的log工具类,可以输出文件名,函数名,行号,也可以输出json格式字符串.


setEnabled 开启或者禁用log默认禁用.

setFilterTag,传入的字符串会显示在日志面板的tag中,便于过滤日志,默认为'xlog'.

setSaveToFile,日志保存在sd卡根目录指定的文件夹中.
